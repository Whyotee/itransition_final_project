# CV Platform

A web app where people log in and fill out a profile made of custom fields.

## What it does so far

- Log in with GitHub or Google
- Create "attributes" (custom fields like "Years of experience", with a type: text, number, date, or yes/no)
- Fill in your own values for those attributes on your profile

## What it's built with

- **Frontend:** React (Vite) with Bootstrap 
- **Backend:** Node.js + Express
- **Database:** PostgreSQL, using Prisma ORM
- **Login:** GitHub and Google (OAuth)
